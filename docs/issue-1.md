# Issue 1 · Searching Strategy and Its Application

The inaugural issue of **GlennMurray Published (GLMR)** collects every midterm submission from the Introduction to AI course under the lecture-note series *Lecture Note in Fundametal AI*. Each PDF is treated as an article with quick links to read online or download the original material.

> **Tip:** “Read online” opens an embedded PDF.js viewer in a new tab. If the viewer fails to load, use the download link and open the file locally.
